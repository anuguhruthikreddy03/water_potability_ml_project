# 💧 Water Potability Prediction App

A Machine Learning web application that predicts whether water is **safe to drink (potable)** or **unsafe**, based on key chemical properties.

Built using **Python, Scikit-learn, and Streamlit**.

---

## 🚀 Features

* 🔍 Predict water potability instantly
* 📊 Displays prediction probability
* 🎛️ Interactive UI with sliders
* ⚡ Fast and lightweight ML pipeline
* 🧠 Uses trained classification model

---

## 🧠 Machine Learning Workflow

1. Data Collection (Water Quality Dataset)
2. Data Preprocessing (Handling missing values, scaling)
3. Feature Engineering
4. Model Training 
5. Pipeline Creation
6. Deployment using Streamlit

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/water-potability-app.git
cd water-potability-app
```

---

### 2️⃣ Create virtual environment (optional)

```bash
python -m venv venv
venv\Scripts\activate   # Windows
```

---

### 3️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

---

### 4️⃣ Run the app

```bash
streamlit run app.py
```

---

## 📊 Input Features

The model uses the following water quality parameters:

* pH
* Hardness
* Total Dissolved Solids
* Chloramines
* Sulfate
* Conductivity
* Organic Carbon
* Trihalomethanes
* Turbidity

---

## 📈 Output

* ✅ **Potable (Safe to Drink)**
* ❌ **Not Potable (Unsafe)**
* 📊 Probability Scores
